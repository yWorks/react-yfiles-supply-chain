# Context Menu
The context menu provides context-specific actions for items.
Set the <TypeLink type="SupplyChain" member="contextMenuItems"/> to populate the context menu with actions.
The <TypeLink type="SupplyChain" /> component introduces a default context menu which provides actions to interact with the chart.

```tsx
function SupplyChainChart() {
  return (
    <SupplyChain data={data} contextMenuItems={SupplyChainContextMenuItems}></SupplyChain>
  )
}
```

To adjust the context menu visualization, add a custom React component using the <TypeLink type="SupplyChain" member="renderContextMenu"/> prop.

<Playground
    code={`
    function App() {
      return (
        <SupplyChain
          data={data}
          renderContextMenu={({item}) => item?.name && (
           <button onClick={() => alert(\`\${item.name} clicked!\`)}>Click here!</button>
          )}
        ></SupplyChain>
      )
    }`
}

        { "name": "Copper-Plate", "id": 2, "parentId": 4 },

        { "name": "Material", "id": 4 }
      ],
      "connections": [{ "sourceId": 1, "targetId": 2 }]
    }`}
/>
